# 🌐 Web Scraping Job Market Analysis

## 📌 Project Overview

This project demonstrates a complete end-to-end workflow of **Web Scraping, Data Cleaning, Exploratory Data Analysis (EDA), and Data Visualization** using Python.

The objective of this project is to collect job listing data from online job portals, transform the raw web data into a structured dataset, and analyze hiring trends across locations, companies, and job roles.

The project showcases practical skills required for Data Analyst and Data Science roles, including data extraction, preprocessing, analysis, and visualization.


# 🎯 Business Problem

Organizations and job seekers often struggle to understand:

- Which locations have the highest job opportunities?
- Which companies hire most frequently?
- What are the most in-demand job roles?
- How are jobs distributed across different locations?
- What hiring patterns exist in the market?

This project aims to answer these questions using scraped job listing data.


# 🎯 Project Objectives

- Extract job listing data from a website using Web Scraping.
- Create a structured dataset from raw HTML content.
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Identify hiring trends and market demand.
- Generate meaningful visualizations.
- Derive actionable business insights.

# 🔄 Project Workflow

## 1️⃣ Web Scraping

- Connected to website using Requests library.
- Parsed HTML using BeautifulSoup.
- Extracted job-related information.
- Collected structured job records.

### Extracted Fields

- Company Name
- Location
- Experience
- Salary
- Skills
- Job Description


## 2️⃣ Data Cleaning

Performed:

- Validation checks
